Guild Wars 2 meets Open Broadcaster Software
An OBS plugin that enhances your livestream with live Guild Wars 2 data

This plugin for Open Broadcaster Software allows you to add live Guild Wars 2 data directly to your stream. This includes various information like your character name, profession or your current location. When this information changes, it will automatically be updated on your stream. More...

In case the included data parsers are not enough, be sure to check the API documentation to see how you can extend it.

Requirements

  • Windows 32-bit or 64-bit (Vista SP2+, 7 SP1+ or 8)
  • Open Broadcaster Software
  • .NET Framework 4.5
  • CLR Host Plugin
